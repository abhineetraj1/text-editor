# Text-editor
This is a simple text editor made in python, using tkinter as a GUI framework

## Features
*	Open text file
*	Create text file
*	Save text file
*	Apply Custom font

## Demo (For Windows 8 | 10 | 11)
*	Download file from [here](http://github.com/abhineetraj1/text-editor/raw/main/app.exe)
*	Run app.exe file
